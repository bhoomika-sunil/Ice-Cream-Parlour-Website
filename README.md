# Ice-Cream-Parlour-Website
A simple and interactive website for an icecream parlour,built using HTML,CSS and Javascript.
Features of my website are it displays flavours and their prices and also ambience and location.
Uses Jaavascript to manage Flavour data dynamically.
Simple CSS styling to improve visual appearence and layout.
